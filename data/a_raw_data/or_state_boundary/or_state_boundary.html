Description:
Abstract:
This theme shows the jurisdictional and cartographic state perimeters for Oregon in the Oregon Lambert Projection.
Purpose: Resource management planning and mapping.
Supplemental_Information:
BLM (Bureau of Land Management), CFF (Cartographic Feature File), DLG (Digital Line Graph), DRG (Digital Raster Graphic), FIPS (Federal Information Processing Standard), GCD (Geographic Coordinate Database layer), GCDB (Geographic Coordinate Data Base), HYD (WODDB Hydrologic Features layer), LLI (Landlines layer), USFS (U.S. Forest Service), USGS (U.S. Geological Survey), WODDB (Western Oregon Digital Data Base). USGS 7.5' quadrangles.

This dataset is held in Librarian and can be exported in various extents, including individual states.

Process_Step:
Process_Description:
BLM contractor used the cited sources (see source information) to extract the desired line work and create the new coverage. This was accomplished in ARCEDIT using the SELECT & PUT commands. The coverage was then cleaned, built for topology and projected to Lambert Conformal Conic. All line work was ultimately snapped to GCDB points where available. The Oregon Geospatial Data Clearinghouse projected the BLM data into the Oregon Lambert Projection. In ArcPlot the Oregon Data was reselected and put into a new coverage.
